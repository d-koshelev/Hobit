# hobit-tools

Future structured tool and action adapter layer for Hobit.

## Belongs Here

- Tool action boundaries.
- Future adapters for terminal, git, database, image, and other capabilities.
- Approval-aware execution interfaces once the contracts are ready.

## Does Not Belong Here Yet

- Real terminal execution.
- Real Git operations.
- Real JDBC or database access.
- Real image editing.
- Widget UI.
